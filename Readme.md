# Contoh Regresi Dummy dengan Python

Repositori ini berisi contoh lengkap **regresi linier dengan variabel dummy** menggunakan Python.  
Cocok digunakan sebagai bahan ajar untuk mata kuliah:

- Pengantar Statistika / Statistika
- Metode Penelitian Kuantitatif
- Ekonometrika / Regresi
- Data Science dasar
